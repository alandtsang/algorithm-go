# Bubble Sort
![bubblesort](https://raw.githubusercontent.com/alandtsang/algorithm-go/master/pic/bubblesort.gif)

English:
- Choose two elements at a time.
- Compare them, large element sink and small element rise.
- Loop n times.

Time complexity O(n^2).

中文：
- 每次选择两个元素
- 进行比较，大元素下沉，小元素上升
- 循环 n 次

时间复杂度 O(n^2)。
